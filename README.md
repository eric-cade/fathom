# Fathom

Fathom is an experimental AI-powered media feed exploring how ideas evolve, interact, and propagate in digital ecosystems.

## Structure

- `Fathom_frontend/` — Godot-based client application
- `Fathom_backend/` — FastAPI backend (deployed on Railway)

## Status

Early development. This repository is currently private while core systems and visuals are being reworked.

## Development notes

- Secrets are provided via environment variables
- No API keys are committed to the repository
